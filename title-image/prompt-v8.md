# Prompt v8 — code position lock, deformation anchored to wax/honey

Iteration after v7. v7 lost the LOTR setting, rendered the cauldron as
patinated bronze instead of black iron, and worst: relocated the C
code from the falling stream onto the crucible's inner wall — a
loophole the model exploited because the code was forbidden everywhere
else. v8 closes that loophole explicitly, anchors the liquid
deformation with concrete visual analogies (honey on vertical glass,
wax candles in a flame), and restores the LOTR-hall environment.
v6 and v7 are preserved unchanged.

```

(1) FOUND DOCUMENTARY PHOTOGRAPH, not CGI render. Asymmetric,
    hand-held, Kodak Vision3 800T film grain, mild halation around
```
